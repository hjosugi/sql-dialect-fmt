# アセットの出所を保存する

最終更新日: 2026-07-11.

## ブランドアートワーク

アイコン、小さなプロモタイル、マルqueeプロモタイルは、このリポジトリのために
組み込みのOpenAI画像生成ワークフローを使用して作成されました。外部のロゴ、スクリーンショット、顧客データ、またはストックアセットは入力として提供されていません。

アイコンのプロンプトは、コードブラケット、整列したフォーマットライン、深いティール、ミント、白の雪の結晶を組み合わせたオリジナルのミニマルエンブレムを要求しました。テキスト、商標、または透かしは含まれていません。
これは、平坦なマゼンタクロマキー背景で生成され、imagegenクロマ除去ヘルパーを使用してアルファに変換され、256、128、48、32、および16ピクセルで検査されました。

プロモのプロンプトは、生成されたエンブレムを唯一の参照として使用し、段階的な抽象ラインが整列したラインになる飽和した開発者ツールアートワークを要求しました。テキスト、SQLコード、サードパーティのロゴ、スクリーンショット、透かしを明示的に禁止しました。最終的なアセットは、正確なChromeウェブストアの寸法にトリミングまたはリサイズされ、ネイティブサイズと半分のサイズで検査されました。

## 製品スクリーンショット

ChromeおよびVS Codeのスクリーンショットは、`source/`の下にあるHTMLフィクスチャの決定論的なブラウザレンダリングです。フィクスチャには、架空のワークスペース名とデモSQLのみが含まれています。拡張ボタン、成功トースト、設定フィールド、サポートされている方言、構文スコープ、およびプライバシー声明は、出荷されたソースコードと一致しています。

## デモビデオ

`chrome/demo-video-1280x720.mp4`は、チェックインされた製品スクリーンショットと決定論的なタイトル/エンドカードから組み立てられた、サイレントの16秒間のH.264/YUV420pプレゼンテーションです。人々、声、顧客データ、サードパーティのロゴ、または著作権で保護された音楽は含まれていません。エンコーディング後に4つの代表的なフレームが視覚的に検査されました。
